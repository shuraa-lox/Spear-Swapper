Fabric loader: 1.18.4+
Mojang Mappings


Mod instructiun:

Bind your keybind in the minecraft controls settings (Default bind - R). U need to have a lunge spear in your hotbar. 
Select the slot with small cooldown, examples:
Good slot: bread, golden_carrot, wind_charge, ender_pearl
Bad slot: mace, sword, axe, pickaxe

Make sure u have enough make sure you have enough hunger/saturation bar. Hold your keybind.
